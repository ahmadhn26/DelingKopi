Image Folder Structure
=====================

This folder contains images for the Kopi Kenangan Senja website.

Structure:
img/
├── menu/
│   ├── kopi-original.jpg
│   ├── kopi-gula-aren.jpg
│   ├── cappuccino.jpg
│   ├── latte.jpg
│   └── default-coffee.jpg (fallback image)
└── products/
    ├── biji-kopi-premium.jpg
    ├── kopi-bubuk.jpg
    ├── tumbler.jpg
    ├── gula-aren.jpg
    └── default-product.jpg (fallback image)

Instructions:
1. Add your coffee menu images to the img/menu/ folder
2. Add your product images to the img/products/ folder
3. Make sure to include default-coffee.jpg and default-product.jpg as fallback images
4. Recommended image size: 300x200 pixels for optimal display
5. Supported formats: JPG, PNG, WebP

Image Naming Convention:
- Use lowercase letters
- Use hyphens (-) instead of spaces
- Use descriptive names (e.g., kopi-mandailing-original.jpg)
- Keep file names short but descriptive

Example file names:
- kopi-mandailing-original.jpg
- cappuccino-mandailing.jpg
- biji-kopi-premium.jpg
- tumbler-exclusive.jpg
